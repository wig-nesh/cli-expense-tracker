# CLI Expense Tracker

## How to Use
- `g++ main.cpp`
- `./a.out`
- The format for entries is `+-amount -d dd-mm-yy -n note with spaces -t tag1,tag2,tag3`
- Date, time and tags are optional. Leaving the date empty will use current date.
